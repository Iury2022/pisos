Crie um app web com HTML, CSS e JavaScript, utilizando a ferramenta Vite para gerar o projeto, pela qual o usuário possa informar a largura e comprimento de uma determinada área plana, além do tipo de piso desejado para cobri-la. Após, quando o usuário clicar no botão "Calcular", o app deve informar qual será a quantidade de piso a ser comprada, além do valor da compra. 

Em seu app, devem ser considerados 3 tipos de piso:

    Cerâmica: R$ 35 o m²
    Porcelanato: R$ 125 o m²
    Madeira: R$ 330 o m²

Importante: o seu app, além de funcional, deve ser devidamente estilizado e responsivo.

Esta avaliação deve ser realizada individualmente. Coloque o seu app em um repositório no Github e envie o link do repositório aqui pelo Moodle até as 23:59 do dia 01/04/2022. Esta atividade vale até 10 pontos.
